# Module Playground

Interactive exploration space for this module.

## Purpose

Use this folder for:
- Quick experiments with module APIs
- Demo scripts showing usage patterns
- Ad-hoc testing during development

## Usage

```bash
# From project root with venv activated
python -m <module_type>.<module_name>.playground.demo
```

## Files

- `demo.py` - Example usage of module features
- Add more `explore_*.py` files as needed

## Note

Playground code is NOT production code. It's for exploration only.
